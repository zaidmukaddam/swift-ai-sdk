title: DeepSeek

```swift
let model = DeepSeekModel("deepseek-chat")
```

Key from `DEEPSEEK_API_KEY`; base URL `https://api.deepseek.com`.

## Features

- Rides the shared chat-completions wire: tools, structured output, and
  vision are encoded for any model that honors them.
- Reasoner models stream `reasoning_content` deltas, surfaced as
  `.reasoningDelta` — no tag parsing needed.
- The `reasoning` parameter maps to `thinking.type` plus
  `reasoning_effort`, where `.xhigh` becomes `max`.
- Prompt-cache hits (`prompt_cache_hit_tokens`) surface as
  `usage.cachedInputTokens`.

## Models

DeepSeek's API serves two aliases — `deepseek-chat` and
`deepseek-reasoner` — that always point at the current generation. The
versions behind them, as of July 2026:
